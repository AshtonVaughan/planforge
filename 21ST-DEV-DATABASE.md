## üé® DESIGN PATTERNS & STYLE NOTES

### Common Tech Stack
- **Framework**: React / Next.js
- **Styling**: Tailwind CSS
- **Base UI**: shadcn/ui
- **Animation**: Framer Motion, GSAP
- **3D**: Spline, Three.js, WebGL/GLSL shaders
- **Icons**: Lucide React

### Popular Visual Styles
1. **Glassmorphism** - Frosted glass effects with backdrop-blur
2. **Gradient animations** - Moving/morphing gradient backgrounds
3. **Glow effects** - Neon/glow borders and highlights
4. **Dark mode first** - Most components are dark-theme by default
5. **Bento grid layouts** - Asymmetric grid card arrangements
6. **Scroll-triggered animations** - Framer Motion intersection observers
7. **Shader backgrounds** - WebGL/GLSL animated backgrounds
8. **Particle effects** - Canvas-based particle systems
9. **Text animations** - Scramble, reveal, typewriter, morphing text
10. **3D effects** - CSS transforms, perspective, Spline scenes

### Common CSS Patterns
```css
/* Glassmorphism */
backdrop-filter: blur(16px);
background: rgba(255, 255, 255, 0.05);
border: 1px solid rgba(255, 255, 255, 0.1);

/* Glow effect */
box-shadow: 0 0 20px rgba(var(--primary), 0.3);

/* Gradient text */
background: linear-gradient(to right, #fff, #999);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;

/* Animated gradient border */
background: linear-gradient(var(--angle), #ff0080, #7928ca, #ff0080);
animation: rotate 3s linear infinite;

/* Dark card with subtle border */
background: hsl(var(--card));
border: 1px solid hsl(var(--border));
border-radius: var(--radius);
```

### Common Tailwind Patterns
```
/* Hero section */
min-h-screen flex items-center justify-center relative overflow-hidden

/* Gradient background */
bg-gradient-to-br from-purple-600 via-pink-500 to-orange-400

/* Glass card */
bg-white/5 backdrop-blur-xl border border-white/10 rounded-2xl

/* Animated hover */
transition-all duration-300 hover:scale-105 hover:shadow-2xl

/* Bento grid */
grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4
```
